# Image Recognition using Teachable Machine and TensorFlow

## Overview
This project demonstrates image classification using a model trained with Google's Teachable Machine and exported to TensorFlow/Keras format.

## Classes
- Dog
- Cat

## Requirements
- Python 3.10+
- TensorFlow 2.12.1
- Pillow
- NumPy
- tf_keras
